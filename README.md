# dropbot-legal
Legal pages for Drop Bot (Luffy Corp)

This repo contains Privacy Policy and Terms of Service pages used for platform verification.

## Files
- index.html
- privacy-policy.html
- terms-of-service.html
- style.css
- robots.txt
- .gitignore

## Publish
Enable GitHub Pages on the `main` branch and root (/). The site will be published at:
`https://<your-username>.github.io/dropbot-legal/`

Use:
- Privacy Policy URL: `/privacy-policy.html`
- Terms of Service URL: `/terms-of-service.html`
