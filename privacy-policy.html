Privacy Policy

Effective date: 2025-11-11
Drop Bot (operated by Luffy Corp). Contact: dropbot.us@gmail.com

1. Summary

Drop Bot facilitates community giveaways on Discord. This policy explains what minimal data is collected, why, how long it is retained, and how you can request deletion.

2. Data we may collect

Discord identifiers
Discord User IDs, Server (Guild) IDs, Channel IDs — required to operate the bot.
Message information
Message IDs and message timestamps for messages participating in giveaways. This enables audit, dispute resolution, and prevents duplicate entries.
Giveaway history
Records of past giveaway messages, winners, and timestamps for up to 7–30 days depending on server configuration.

3. Purpose & legal basis

We only collect the data necessary to run giveaways and provide dispute/audit logs to the server owner. The legal basis is legitimate interest: to operate and secure the service, prevent abuse, and provide accurate results.

4. Retention

Data is retained for a maximum of 30 days by default. Typical retention is 7 days. Server owners may request longer or shorter retention; stored data is removed automatically when expired.

5. Who has access

Only the bot and the server owner (as configured) can see giveaway result messages and logs. No third-party analytics or marketing tracking is performed by Drop Bot.

6. Third parties & payments

Drop Bot does not process payments. Any payments or prizes are handled by server owners independently, not via Drop Bot. If you choose to pay a third-party service connected to a server, please review that service's privacy policy.

7. Security

Secrets such as API tokens are never stored in public repositories. We follow basic security practices: environment variables for tokens, restricted access, and minimal storage duration.

8. Your rights

You may request access, correction, or deletion of personal data stored by Drop Bot by contacting dropbot.us@gmail.com. Removal requests targeting server-level stored data must be routed via the server owner where appropriate.

9. Changes to this policy

We may update this document. The effective date at the top will be updated. For material changes, we will provide a notice on our support contact.
